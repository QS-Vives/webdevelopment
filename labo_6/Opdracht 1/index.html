Css:

li { display:inline-block; width:150px; } li > a { color:red; }

Dit zal wel toegepast worden op elke ul. Indien dit niet gewenst is kan er nog een andere manier toegepast worden: Je geeft de ul een class, bijvoorbeeld "menu", dan kun je volgende css gebruiken:

.menu > li { display:inline-block; width:150px; } .menu > li > a { color:red; }
